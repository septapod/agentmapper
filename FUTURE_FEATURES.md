# AgentMapper - Future Features

Potential enhancements identified from original design specifications.
These are documented for future reference - not scheduled for immediate implementation.

---

## Real-Time Collaboration

Enable multiple participants to work together simultaneously.

- [ ] WebSocket/Supabase Real Time for live sync
- [ ] Participant cursor presence (see where others are)
- [ ] User avatars and presence indicators
- [ ] Live sticky note updates across participants
- [ ] Real-time vote aggregation

---

## Role-Based Access Control

Different permissions and views based on user role.

- [ ] Facilitator role (leads sessions, can cluster notes, delete others' notes)
- [ ] Participant role (adds notes, votes, comments)
- [ ] Observer role (read-only access)
- [ ] Admin role (manages org settings, billing)
- [ ] Role-specific UI views and permissions

---

## Persistent Governance Sidebar

Always-visible governance guidance during sessions.

- [ ] Always-visible governance checklist during sessions
- [ ] Red flag indicators for missing required items
- [ ] Links to NIST AI RMF / ISO 42001 documentation
- [ ] Blocking items that prevent progression until addressed

---

## Facilitator Tools

Enhanced controls for session facilitators.

- [ ] Session timer with Pomodoro or custom intervals
- [ ] Presenter mode (enlarged view, speaker notes)
- [ ] Cluster/group notes tool (drag multiple into groups)
- [ ] Pin notes to keep visible while scrolling
- [ ] Live transcription capture toggle
